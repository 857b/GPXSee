# GPXSee
GPX viewer and analyzer.

* User-definable map sources.
* Track and elevation/speed graphs.
* Support for multiple tracks in one view.
* Support for POI files.
* Export to PDF.
* Native GUI for Windows, Mac OS X and Linux.

## Build
### Linux/OS X
```shell
lrelease gpxsee.pro
qmake gpxsee.pro
make
```
### Windows
```shell
lrelease gpxsee.pro
qmake gpxsee.pro
nmake release
```
